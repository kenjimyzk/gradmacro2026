# マクロ経済学B シラバス案

## 講義概要
本講義では、現代マクロ経済学の中核であるニューケインジアン・マクロ経済学を学びます。リアル・ビジネス・サイクル（RBC）モデルから始まり、価格・賃金の硬直性、独占的競争、金融政策の理論的基礎を段階的に習得します。特に、金融政策のルール、裁量とコミットメント、異質な家計を含むモデル（TANKモデル）など、最先端の理論と政策分析手法を学びます。

## 到達目標
1. ニューケインジアン・マクロ経済学の理論的枠組みを理解する
2. 動学的最適化と対数線形近似の手法を習得する
3. 金融政策の分析手法（決定性、裁量とコミットメント）を身につける
4. 最新の理論モデル（TANKモデル、ニューケインジアンクロス）を理解し応用できる

- キャリブレーションと数値シミュレーション

### 第4回：政府の役割
