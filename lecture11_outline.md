# 第11回：価格硬直性をもつ TANK モデル 講義アウトライン

## 講義の位置づけ

第10回では、代表的家計の RANK モデルに賃金硬直性を加えた。そこで重要だったのは、賃金所得と配当所得の分解である。RANK では同じ家計が賃金所得と配当を受け取るため、この分解は集計消費には出にくい。しかし TANK では、賃金を受け取る家計と配当を受け取る家計が異なるため、所得分配そのものが金融政策の伝播を変える。

第11回では、賃金硬直性はいったん外し、価格硬直性だけを残す。これは、第10回の複雑さをそのまま持ち込むのではなく、家計異質性による新しい伝播経路を単独で見るためである。第10回が「名目硬直性を増やす回」だったのに対し、第11回は「代表的家計を2タイプの家計に分ける回」と位置づける。

先に結論を言うと、RANK では集計需要を代表的家計の Euler 方程式で表せたが、TANK ではそれができない。貯蓄家計の Euler 方程式に加えて、手元流動性制約家計の現在所得制約が集計需要を決める。

## 第10回から第11回への接続

| 論点 | 第10回：RANK with dual rigidity | 第11回：TANK with price rigidity |
|---|---|---|
| 家計 | 代表的家計 | 貯蓄家計と手元流動性制約家計 |
| 新しい摩擦 | 賃金硬直性 | 家計異質性 |
| 価格硬直性 | あり | あり |
| 賃金硬直性 | あり | なし |
| 賃金フィリップス曲線 | あり | なし |
| 集計需要 | 代表的家計の Euler 方程式 | Type S の Euler 方程式と Type H の現在所得制約 |
| 所得分配 | 集計消費には出にくい | 金融政策伝播の中心 |
| 追加変数 | $\mu^w_t,\pi^w_t$ | $c^H_t,c^S_t,d_t,z_t$ |

第11回の導入では、次の対比を強調する。

1. 第10回では、賃金硬直性により「実質賃金がすぐには調整しない」ことが問題だった。
2. 第11回では、賃金が柔軟でも「所得を受け取る家計が誰か」によって集計需要が変わる。
3. 第10回で確認した配当式
   $$
   d_t=y_t-(w_t+n_t)
   $$
   は、RANK では補助的な式だったが、TANK では Type S 家計の消費を動かす中心式になる。

## 講義の目的

第11回の到達目標は次の5点である。

1. TANK モデルの2タイプ家計、Type H と Type S の役割を理解する。
2. RANK の代表的家計 Euler 方程式が、TANK では Type S 家計の Euler 方程式に置き換わることを理解する。
3. Type H 家計の消費が現在所得に結びつくため、賃金所得、移転、配当の分配が集計需要を動かすことを理解する。
4. 価格フィリップス曲線は第7回・第10回と同じ価格ブロックから来ることを確認する。
5. $\lambda=0$、または Type H 家計を取り除くと、価格硬直性だけの RANK モデルに戻ることを確認する。

## 推奨する講義構成

### 1. 導入：第10回で見えた「分配」の入口

最初に、第10回の最後で出てきた論点を復習する。

- RANK では、賃金所得と配当は同じ代表的家計に戻る。
- したがって、労働所得と配当の分解は集計消費には出にくい。
- TANK では、Type H は主に労働所得と移転で消費し、Type S は債券と企業所有から影響を受ける。
- そのため、金融政策は実質利子率だけでなく、賃金、雇用、配当、移転を通じて伝播する。

ここで第11回の中心メッセージを置く。

$$
\text{RANK: } c_t \text{ is pinned down by one Euler equation.}
$$

$$
\text{TANK: } c_t=\lambda c^H_t+(1-\lambda)c^S_t
\text{ and the two consumptions move for different reasons.}
$$

### 2. 経済環境：2タイプの家計

人口のうち $\lambda$ が手元流動性制約家計 Type H、残りの $1-\lambda$ が貯蓄家計 Type S である。

- Type H 家計は債券を保有せず、企業も所有しない。
- Type H 家計の消費は現在の可処分所得に等しい。
- Type S 家計は名目債券を取引し、企業を所有する。
- Type S 家計の Euler 方程式が資産価格と実質利子率を決める。
- 企業配当は Type S 家計に帰属する。

第10回との違いは、代表的家計の1本の予算制約を2本に分ける点である。第10回で増えた賃金フィリップス曲線はここでは使わない。

### 3. 家計ブロック

集計消費は
$$
c_t=\lambda c^H_t+(1-\lambda)c^S_t
$$
である。

Type S 家計の Euler 方程式は
$$
r_t=\gamma\mathbb{E}_t\Delta c^S_{t+1}
$$
である。ここで重要なのは、右辺が集計消費 $c_t$ ではなく、Type S の消費 $c^S_t$ である点である。

Type H 家計は現在所得を消費する。政府支出を捨象し、移転を $z_t$ と書くと、直観的には
$$
c^H_t
\quad\text{is tied to}\quad
w^H_t+n^H_t+z_t
$$
である。

Type S 家計は、労働所得に加えて配当を受け取り、Type H への移転を負担する。したがって、直観的には
$$
c^S_t
\quad\text{is tied to}\quad
w^S_t+n^S_t+d_t-\frac{\lambda}{1-\lambda}z_t
$$
である。

この段階では係数を細かく覚えさせるよりも、次の符号を理解させる。

- $z_t$ は Type H の消費を押し上げる。
- $z_t$ は Type S の可処分所得を押し下げる。
- $d_t$ は Type S の可処分所得を押し上げる。
- $\lambda$ が大きいほど、Type H の現在所得チャネルが集計需要で重要になる。

### 4. 労働市場：柔軟賃金に戻す

第11回では、第10回の賃金硬直性をいったん外す。したがって、賃金フィリップス曲線
$$
\pi^w_t=\beta\mathbb{E}_t\pi^w_{t+1}-\kappa_w\mu^w_t
$$
は出てこない。

必要なのは、柔軟賃金の条件である。タイプ別の労働供給を使うなら、
$$
\mu^H_t
=w^H_t-\gamma c^H_t-\varphi n^H_t=0,
\qquad
\mu^S_t
=w^S_t-\gamma c^S_t-\varphi n^S_t=0
$$
である。

この式は、第10回の賃金マークアップをタイプ別に分け、さらに柔軟賃金なのでゼロに置いたものと読める。

労働パッカーを残す場合は、タイプ別労働需要を
$$
n^H_t=n_t-\psi_w(w^H_t-w_t),
\qquad
n^S_t=n_t-\psi_w(w^S_t-w_t)
$$
と書く。講義では、細かい導出は短くし、Type H と Type S の消費が異なるため、柔軟賃金でもタイプ別の賃金・労働が動きうる点を強調する。

### 5. 企業・価格設定ブロック

価格ブロックは第7回・第10回と同じである。$\alpha=0$ なら生産関数は
$$
y_t=a_t+n_t
$$
である。

実質限界費用を $mc_t=w_t-a_t$ と書くと、価格フィリップス曲線は
$$
\pi^p_t
=
\beta\mathbb{E}_t\pi^p_{t+1}
+
\kappa_p mc_t
$$
である。

価格マークアップ表示を使えば
$$
\pi^p_t
=
\beta\mathbb{E}_t\pi^p_{t+1}
-
\kappa_p\mu^p_t,
\qquad
\mu^p_t=a_t-w_t.
$$

第10回では、これに賃金フィリップス曲線が加わった。第11回では、価格フィリップス曲線は残るが、賃金フィリップス曲線は柔軟賃金条件に置き換わる。

配当は
$$
d_t=y_t-(w_t+n_t)
$$
である。第10回ではこの式は「価格硬直性だけの場合と賃金硬直性だけの場合の比較」に出てきた。第11回では、この配当が Type S 家計に入るため、集計需要の一部になる。

### 6. 金融政策と集計需要

事前実質利子率は
$$
r_t=r^N_t-\mathbb{E}_t\pi^p_{t+1}
$$
である。

標準的なテイラー・ルールを使うなら
$$
r_t=\phi \pi^p_t-\mathbb{E}_t\pi^p_{t+1}-m_t
$$
と書ける。

RANK では、実質利子率の上昇は代表的家計の Euler 方程式を通じて集計消費を直接下げた。TANK では、実質利子率が直接動かすのは Type S の消費である。Type H の消費は、金融政策が賃金、雇用、移転、配当に与える影響を通じて動く。

ここで、金融政策の伝播経路を次のように整理する。

1. 実質利子率チャネル：$r_t$ が $c^S_t$ を動かす。
2. 現在所得チャネル：$w^H_t+n^H_t$ が $c^H_t$ を動かす。
3. 配当チャネル：$d_t$ が $c^S_t$ を動かす。
4. 再分配チャネル：$z_t$ が $c^H_t$ と $c^S_t$ を反対方向に動かす。

### 7. 方程式リスト

講義ノートでは、最初に簡略版、次に必要なら拡張版を置く。

簡略版の中心方程式は次の通りである。

$$
\begin{aligned}
c_t
&=
\lambda c^H_t+(1-\lambda)c^S_t,
\\
r_t
&=
\gamma\mathbb{E}_t\Delta c^S_{t+1},
\\
c^H_t
&=
\text{current income of Type H},
\\
c^S_t
&=
\text{labor income, dividends, and transfers of Type S},
\\
y_t
&=
a_t+n_t,
\\
y_t
&=
c_t,
\\
\pi^p_t
&=
\beta\mathbb{E}_t\pi^p_{t+1}
+
\kappa_p mc_t,
\\
d_t
&=
y_t-(w_t+n_t),
\\
r_t
&=
\phi\pi^p_t-\mathbb{E}_t\pi^p_{t+1}-m_t.
\end{aligned}
$$

このリストの後に、「第7回との違い」と「第10回との違い」を並べると見通しがよい。

### 8. RANK への縮約

第11回の最後に、整合性チェックとして RANK への縮約を確認する。

Type H 家計が存在しない、つまり
$$
\lambda=0
$$
なら
$$
c_t=c^S_t
$$
となる。このとき Type S の Euler 方程式は
$$
r_t=\gamma\mathbb{E}_t\Delta c_{t+1}
$$
となり、価格硬直性だけの RANK モデルに戻る。

この確認により、第11回の TANK モデルは、第7回の RANK モデルを壊しているのではなく、家計異質性を追加した拡張であることを示せる。

### 9. 第10回との比較まとめ

講義の終盤で、次の表を置く。

| 比較 | 第10回 | 第11回 |
|---|---|---|
| 何を追加したか | 賃金硬直性 | 家計異質性 |
| 何を固定したか | 代表的家計 | 柔軟賃金 |
| 中心となる新変数 | $\mu^w_t,\pi^w_t$ | $c^H_t,c^S_t,d_t,z_t$ |
| 主要な新しい式 | 賃金フィリップス曲線 | Type H の現在所得制約 |
| 政策トレードオフ | 価格インフレ率、賃金インフレ率、需給ギャップ | 実質利子率チャネルと所得分配チャネル |
| 次への橋渡し | 賃金硬直性の役割 | TANK に賃金硬直性を戻す |

第10回で増やした賃金硬直性と、第11回で増やす家計異質性は、別々の拡張である。次に二重の名目硬直性をもつ TANK を扱うなら、この2つを合わせればよい。

## 演習問題案

**問1：RANK への縮約**

TANK モデルで $\lambda=0$ とする。このとき集計消費と Euler 方程式が価格硬直性だけの RANK モデルに戻ることを示しなさい。

**問2：Type H の現在所得チャネル**

Type H 家計の消費が現在の労働所得に等しいとする。金融引き締めが雇用を下げるとき、Type H の消費はどのように動くか説明しなさい。

**問3：配当チャネル**

配当が Type S 家計に帰属するとする。賃金支払いが増えると配当はどう動くか。また、それは Type S の消費にどのような影響を与えるか説明しなさい。

**問4：再分配ショック**

Type H への移転 $z_t$ が増えると、Type H と Type S の消費はそれぞれどう動くか。集計消費への効果が $\lambda$ に依存する理由を説明しなさい。

**問5：第10回との比較**

第10回では賃金インフレ率が政策問題に入った。第11回では賃金インフレ率が登場しない。それにもかかわらず、第11回で金融政策の伝播が RANK と異なる理由を説明しなさい。

- 賃金硬直性は扱わないことを明示する。これは省略ではなく、家計異質性の効果を分離するための設計である。
- 第10回の記号 $mc_t$、$\mu^p_t$、$d_t$ との整合性を保つ。
- 配当 $d_t$ と移転 $z_t$ は、誰の予算制約に入るかを毎回明示する。
- 補論では、係数つきの正確なログ線形予算制約を置くとよい。本編では、まず符号とメカニズムを優先する。
